# PRODIGY_WD_2
⏱️ Stopwatch Web Application

This project is a Stopwatch Web Application developed using HTML, CSS, and JavaScript. It was created as Task-02 for the Web Development Internship at Prodigy InfoTech.

📌 Project Description

The Stopwatch Web Application allows users to accurately measure and record time intervals. The application includes features such as starting, pausing, resetting the stopwatch, and recording lap times. It provides a simple and interactive user interface for tracking time efficiently.

🚀 Features

Start the stopwatch

Pause the stopwatch

Reset the stopwatch

Record and display lap times

Real-time time tracking

User-friendly interface

🛠️ Technologies Used

HTML5 – Structure of the application

CSS3 – Styling and layout

JavaScript – Functionality and time tracking logic

📂 Project Structure

index.html – Main structure of the application

style.css – Styling of the stopwatch interface

script.js – JavaScript logic for stopwatch functionality

🎯 Learning Outcomes

Understanding JavaScript timing functions

Handling user interactions and events

Building interactive web applications
